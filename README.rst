Example Usage
-------------


``$ service mattd start``

Starting Matt Daemon... [ OK ]

``$ service mattd status``

Matt Daemon is running.

``$ service mattd stop``

You can't stop Matt Daemon.  Permission denied.  [FAILED]

``$ sudo service mattd stop``

Stopping Matt Daemon... [ OK ]
